## Run virtual env

1. On MacOS:

```bash
source .env/bin/activate
```

2. On Windows:

```powershell
.\.env\Scripts\Activate.ps1
```

## Install dependencies python 3.x

```bash
pip install requirements.txt
```

## Run example

Use python command as your enviroments

```bash
python main.py
```

## Output

Output folder is ./outputs
